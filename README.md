# 💸 KalBudget

KalBudget is a full-stack expense tracking web application that helps users manage their finances, track spending, and visualize budgets in a clean and intuitive interface.

---

## 🚀 Live Demo
🌐 https://kalbug.vercel.app/

---

## 📌 Features

- 🔐 User Authentication (Login / Signup via Backendless)
- 💰 Expense Tracking & Categorization
- 📊 Budget Dashboard with insights
- ⚡ Real-time data updates
- 🎯 Monthly budget management
- 🤝 Split expenses with friends (designed for trips, shared living, and group expenses)
- 📱 Clean and responsive UI

---

## 🛠️ Tech Stack

**Frontend**
- Next.js
- React
- Tailwind CSS

**Backend**
- Backendless (API, Database & Authentication)

**Other Tools**
- AI tools (for debugging & development assistance)
